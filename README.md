# Task Management API

Production-style but interview-defendable FastAPI backend for task management.

The main implementation lives in [backend/README.md](backend/README.md). Use this root README for the fastest path:

1. Copy [backend/.env.example](backend/.env.example) to `backend/.env`.
2. Run Docker Compose from the `backend/` folder.
3. Open `http://localhost:8000/docs`.
